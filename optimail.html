Project Description

The problem now for Canada Post mail carriers is planning switching to a new postal transformation system, carriers would have heavy workloads that will take longer to complete in addition to paperwork. Therefore, as a team we created an all-in-one mobile application that provides letter carriers with all the necessary tools to deliver mail efficiently and accurately.
